# Knox Barbell - Gym Website

![Knox Barbell Banner](https://via.placeholder.com/1200x400?text=Knox+Barbell+Gym) <!-- Replace with actual banner image -->

A modern, responsive front-end website developed for Knox Barbell gym located in Bhairahawa, Nepal. Built with pure HTML, CSS, and JavaScript.

## 🏋️‍♂️ Project Overview

Knox Barbell is a premium gym facility in Bhairahawa, Nepal specializing in strength training and bodybuilding. This website was commissioned to serve as their digital presence, showcasing facilities, trainers, programs, and membership options.

### Key Features Implemented:
- Fully responsive design (mobile, tablet, desktop)
- Interactive event integration
- Newsletter subscription integration
- Online membership applications
- Real-time announcements section
- Trainer profile showcases
- Service pricing displays
- Contact form with client-side validation
- Smooth UI animations and transitions

## 🌟 Key Sections

- **Home**: Engaging hero section with conversion-focused CTA
- **About**: Brand story and facility highlights
- **Services**: Detailed training programs
- **Events**: Competition schedules and registrations
- **Trainers**: Professional profiles with specialties
- **Membership**: Tiered pricing structure
- **Contact**: Location mapping and inquiry forms
